<h1 align="center">Snake 🐍</h1>
<p align="center"><em>A terminal-based snake game built with C</em></p>

## Run the Game
```bash
gcc -o snake main.c
./snake
```

## Controls
* `Esc`: Quit
* `W`: Move up
* `S`: Move down
* `A`: Move left
* `D`: Move right
